Developer platform

Developers

Use the UI or integrate with SDK/Gateway. Every SDK call is workspace-scoped and idempotency-aware.

TypeScript SDK

npm install @angorapay/sdk

await angora.runAgentMission({
  agent: 'prediction_market',
  mission: 'Check whether this BTC prediction market is mispriced.',
  maxSpendUSDC: '0.05',
  requiredProof: true,
  idempotencyKey: 'mission-001-agent-run-001'
});

API endpoint

POST /v1/angora/agent-missions/run
Authorization: Bearer ang_live_xxxxx
Idempotency-Key: mission-001-agent-run-001
